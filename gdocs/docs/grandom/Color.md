## Color
`func Color(type int) string`

Randomly generate colors, `RGB`, `RGBA`, `COLOR`, `HEX`:

```js
grandom.Color(C_RGB)    // => rgb(247,39,65)
grandom.Color(C_RGBA)   // => rgba(124,236,25,0.55)
grandom.Color(C_HEX)    // => #c5e814
grandom.Color(C_COLOR)  // => Red
```
